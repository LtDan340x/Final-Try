Race App

Carbon fiber race control with multiview racer access, lane calls, random pairings, one-time bye protection, and track-ready display screens.

Open Admin Control Open Racer View Open MultiView Board

What this build includes

  • Random round pairing
  • Random lane assignment
  • Bye only once per racer unless reset
  • Call-to-lanes alert banner
  • Admin, Racer, and Board views
  • Netlify Functions + Blobs support
  • Local fallback mode for testing

How to deploy

  1. Drag this folder into a new GitHub repo or upload to Netlify.
  2. Deploy on Netlify.
  3. Open /admin.html to manage racers.
  4. Share /racer.html or QR code with racers.
  5. Use /board.html on a TV or tablet.
